# Theme Notes

## Previous Direction

The prior DealState aesthetic was a dark-first dusk institutional system:

- blue-hour estuary hero image
- dark navy hero scrim
- glass-like ivory product vignette
- map-line and radar-style background treatment
- teal and green operational accents
- gold/copper caution accents
- explicit `Dusk`/`Light` theme toggle in the top nav

It framed DealState as a deal-state observatory and investment workspace. The strongest retained ideas were:

- source-backed state as the product centre
- Project Nova as the main theatre surface
- visible conflicts, missing material and latest changes
- three-column deal-room structure
- deterministic synthetic data boundary

## Why It Was Replaced

The Humanist Compute Atelier rebuild changes the emotional register from dark observatory/dashboard to archival state register:

- paper canvas instead of dark hero image
- editorial serif scale instead of dark SaaS contrast
- ledgers, marginalia, slips and provenance marks instead of observatory graphics
- a single canonical theme instead of light/dark switching
- mobile casebook flow instead of compressed desktop dashboard
